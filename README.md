# Zenthink | Providing Perpetual IT Solutions

## 📖 Overview

The official corporate website for **Zenthink** (https://www.zenthink.in). 
This repository contains the source code for a modern, high-performance web application designed to showcase Zenthink's comprehensive IT solutions, services, and innovative approaches to driving business growth. 

Built with the latest web technologies, the site features smooth animations, responsive design, and a highly optimized user experience.

## ✨ Features

- **Modern Tech Stack**: Built on the robust **Next.js 15** App Router with **React 19**.
- **Stunning UI/UX**: Styled completely with **Tailwind CSS v4** and customized Radix UI components for accessibility.
- **Fluid Animations**: Integrated with **Framer Motion** and **Lenis** for buttery-smooth scrolling and entrance animations.
- **Responsive Design**: Flawless experience across all devices, from mobile to desktop.
- **Contact & Communication**: Automated email handling via **Nodemailer**.
- **SEO Optimized**: Fully compliant with modern SEO standards ensuring maximum visibility.

## 🛠️ Technology Stack

- **Framework**: Next.js 15 (Turbopack)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion, Lenis (Smooth Scroll)
- **UI Components**: Radix UI, Lucide React, React Icons
- **Backend/API**: Nodemailer (Email services)

## 🚀 Getting Started

Follow these steps to run the project locally.

### Prerequisites

Make sure you have Node.js installed on your machine. We recommend using `npm` or `yarn`.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/zenthink.git
   cd zenthink
   ```

2. Install the dependencies:
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. Set up your environment variables:
   Create a `.env.local` file in the root directory and add the necessary variables (e.g., mail credentials for Nodemailer).

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
